Linux Process Inspector

A lightweight system monitoring tool for Linux that inspects running processes and displays live system metrics such as CPU usage and RAM consumption.

This project was built to provide a simple, fast, and minimal alternative to heavy system monitoring tools. It continuously updates process information in real time, allowing users to quickly observe system performance and resource usage.

Features

📊 Live CPU usage monitoring

🧠 Real-time RAM usage tracking

🔄 Continuous process updates

⚡ Lightweight and fast

🐧 Designed specifically for Linux systems
